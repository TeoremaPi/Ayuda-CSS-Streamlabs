Configurar el chat

Para evitar problemas en la ventana del chat los mensajes aparecen con una tipografía estándar, en color negro y con fondo transparente. No obstante, las características del texto pueden modificarse para adecuarse al layout de cada programa.

Con esta herramienta podrás modificar facilemnte el formato del texto del chat. Juega con los diferentes parámetros y obtendrás el código CSS personalizado que posteriormente deberás introducir en la ventana correspondiente de SteamlabsOBS (ver imagen superior).

Puedes hacerte una idea del aspecto que tendrá el chat en la ventana de previsualización pero debes marcar las casillas de verificación para que las modificaciones se incluyan en el código.
